Sliding doors technique for fancy buttons.

CSS sliding doors.

By using sliding doors technique, you can apply the style to any link and the same time you do not have to create an image for each button.

What is sliding doors technique?

The way to strech the button-width horizontally. To accomplish this you have to craete two background images that will define the button: one for the left side and one for the right side.

The smaller right image will slide on top of the larger, left image. (that is why it is called sliding doors.) The more the right image slides to the left, the narrower button will be and vice versa.

Styling the button

You will have span element within the a element. The span tag contains left, wider image and text. The width of the text will determine the size of the buttons. Inspect to see the code
